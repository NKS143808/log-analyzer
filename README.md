# 📄 Log Analyzer

A Python tool that reads a log file and summarizes the number of **ERROR**, **WARNING**, and **INFO** entries.

## Features

- Read log files
- Count ERROR entries
- Count WARNING entries
- Count INFO entries
- Display a summary

## Technologies Used

- Python
- collections.Counter

## How to Run

```bash
py main.py
```

## Example

```
Enter log file name: system.log

Log Summary
------------------------------
Errors   : 4
Warnings : 2
Info     : 25
```
